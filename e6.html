RP1 - Accès VAR et Rapports de Match

Fiche d'identité

  • Réalisation n° 01
  • Période : mars 2026
  • Lieu : IRIS École supérieure d'informatique, Paris 17e
  • Modalité : seul
  • Intitulé : accès VAR et rapports de match

Objectif

Mettre à disposition un accès réseau sécurisé, isolé et performant pour les arbitres afin de consulter les flux VAR et déposer leurs rapports de match dans un espace réseau dédié.

Résultats attendus

  • Disponibilité du SSID Referee-Wifi sur le VLAN 50 pour le visionnage des flux VAR sans coupure
  • Comptes Active Directory Arbitre opérationnels avec désactivation automatique via PowerShell en fin de match
  • Montage automatique du lecteur réseau R: sur les sessions arbitres grâce à une GPO
  • Isolation complète du VLAN 50 vis-à-vis du reste de l'infrastructure via filtrage sur le pare-feu

Ressources utilisées

  • Matériel : Cisco 2821, Catalyst 2960, borne AIR-CAP3502I, serveur DELL PowerEdge R740
  • Virtualisation : VMware vSphere ESXi 8.1
  • Systèmes : Windows Server 2022, Windows 11, Debian 12
  • Outils : PuTTY, Wireshark, OCS/GLPI, Zimbra, Snort
  • Documents : contexte Stadium, cahier des charges, environnement technologique d'apprentissage

Description de la réalisation

J'ai déployé une infrastructure système et réseau dédiée aux officiels de match. La solution repose sur un VLAN 50 en 172.20.5.0/24 et sur le SSID Referee-Wifi. La gestion des utilisateurs est centralisée sur un contrôleur de domaine Active Directory avec automatisation des accès par GPO et par script PowerShell.

Étapes de mise en place

  • Création du VLAN 50 sur le switch et configuration du routage avec ACL d'isolation
  • Configuration de la borne Wi-Fi avec le SSID Referee-Wifi taggé sur le VLAN 50
  • Création des comptes Arbitre dans l'Active Directory
  • Mise en place d'un partage de fichiers sécurisé pour les rapports
  • Déploiement d'une GPO pour mapper automatiquement le lecteur R:
  • Création d'un script PowerShell pour désactiver les comptes après le match

Éléments techniques clés

VLAN 50 172.20.5.0/24 ACL / filtrage Active Directory GPO PowerShell Wi-Fi 802.1Q Partage réseau

RP2 - Support de Match et Communication

Fiche d'identité

  • Réalisation n° 02
  • Période : mars 2026
  • Lieu : IRIS École supérieure d'informatique, Paris 17e
  • Modalité : seul
  • Intitulé : support de match et communication

Objectif

Garantir le maintien en condition opérationnelle de l'environnement déployé pendant les matchs, assurer le suivi des incidents, l'inventaire des tablettes VAR et la communication officielle avec les arbitres.

Résultats attendus

  • Supervision temps réel de la borne Referee-Wifi et des serveurs critiques via Nagios
  • Création automatique d'un ticket GLPI lors d'une panne détectée par la supervision
  • Inventaire centralisé de 100 % des tablettes VAR via OCS Inventory et GLPI
  • Mise en service de Zimbra pour les convocations et informations techniques
  • Traçabilité complète des incidents et interventions dans GLPI

Ressources utilisées

  • Matériel : Cisco 2821, Catalyst 2960, borne AIR-CAP3502I, serveur DELL PowerEdge R740
  • Virtualisation : VMware vSphere ESXi 8.1
  • Systèmes : Windows Server 2022, Windows 11, Debian 12
  • Solutions déployées : Nagios, GLPI, OCS Inventory, Zimbra
  • Outils complémentaires : Snort, PuTTY, Wireshark

Description de la réalisation

Après le déploiement de l'accès réseau dans la RP1, j'ai mis en œuvre une plateforme de gestion et de surveillance centralisée afin d'assurer le MCO de l'infrastructure. J'ai supervisé les équipements réseaux, automatisé la gestion des incidents, organisé l'inventaire du parc mobile et installé une solution de messagerie collaborative dédiée aux officiels.

Composants mis en place

  • Supervision avec Nagios pour suivre la borne Wi-Fi et les serveurs critiques avec alertes
  • Helpdesk GLPI avec historique des incidents et des modifications techniques
  • Inventaire automatisé via OCS Inventory pour remonter les caractéristiques des tablettes VAR
  • Messagerie collaborative Zimbra pour les boîtes mail et les listes de diffusion des officiels
